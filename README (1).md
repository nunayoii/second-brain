## 📖 專案簡介

「魯迅·第二大腦」是一個創新的數位人文（Digital Humanities）專案，旨在透過人工智慧技術，讓使用者能夠與中國現代文學巨匠魯迅（1881-1936）進行深度對話。本專案不僅是一個技術實踐，更是對文學傳承與數位化轉型的探索。

### 專案背景

在數位時代，如何讓經典文學作品與思想家的智慧延續並觸及更多人？本專案嘗試運用大型語言模型（LLM）技術，構建一個「數位魯迅」，使用者可以：

- 📚 系統性地瀏覽魯迅的完整著作目錄
- 💬 以自然語言與「魯迅」進行對話
- 🎭 體驗獨特的「狂人模式」，感受《狂人日記》中覺醒者的視角

### 核心理念

> 「橫眉冷對千夫指，俯首甘為孺子牛。」——魯迅

本專案希望透過技術手段，讓魯迅的批判精神與人文關懷在數位時代得以延續，為文學教育和研究提供新的可能性。

## ✨ 功能特色

### 1. 完整作品資料庫

系統性整理魯迅一生的文學創作，包含：

| 類別 | 作品數量 | 代表作品 |
|------|----------|----------|
| 小說集 | 3 部 | 《吶喊》、《彷徨》、《故事新編》 |
| 散文集 | 2 部 | 《朝花夕拾》、《野草》 |
| 雜文集 | 12 部 | 《熱風》、《華蓋集》、《且介亭雜文》等 |
| 學術著作 | 2 部 | 《中國小說史略》、《漢文學史綱要》 |
| 翻譯作品 | 多部 | 果戈里《死魂靈》、法捷耶夫《毀滅》等 |

### 2. AI 驅動的對話系統

- **角色扮演**：AI 模型經過特殊提示詞工程，能夠模擬魯迅的語言風格、思維方式和價值觀
- **上下文記憶**：支援多輪對話，系統會記住對話歷史，實現連貫的交流
- **串流回應**：採用串流技術，即時顯示 AI 的回應過程

### 3. 狂人模式 🌙

靈感來自魯迅的代表作《狂人日記》（1918），這是中國現代文學史上第一篇白話文小說。

**開啟狂人模式後：**

- 🎨 **視覺變化**：背景轉為深紅色漸變動畫，營造詭異氛圍
- 🧠 **人格切換**：AI 將以「狂人」的視角回應——敏感、多疑、卻又清醒
- 📝 **語言風格**：跳躍式的思維、充滿隱喻的表達、對「吃人」社會的批判

### 4. 響應式設計

- 📱 完美適配手機、平板、桌面設備
- 🌓 自動偵測系統深色/淺色模式偏好
- ♿ 注重無障礙設計，支援鍵盤操作

## 🛠 技術架構

### 前端技術棧

```
├── HTML5          # 語義化標記
├── CSS3           # 自訂屬性、Flexbox、動畫
├── JavaScript     # ES6+ 原生 JavaScript
└── External Libraries
    ├── Marked.js      # Markdown 解析
    ├── Google Fonts   # 思源宋體、站酷小薇
    └── Font Awesome   # 圖標庫
```

### AI 整合

本專案整合 **Poe Embed API**，使用 **Claude Sonnet 4.5** 作為對話引擎：

```javascript
// 核心 API 調用示例
await window.Poe.sendUserMessage(
    `@Claude-Sonnet-4.5 ${systemPrompt}\n\n${userMessage}`,
    {
        handler: 'luxun-handler',
        stream: true,        // 串流回應
        openChat: false      // 在應用內顯示
    }
);
```

### 設計系統

採用自訂 CSS 變數實現一致的視覺語言：

```css
:root {
    --bg-primary: #1a1612;      /* 古籍褐色 */
    --accent: #c4956a;          /* 金銅色點綴 */
    --madman-red: #8b0000;      /* 狂人模式主色 */
}
```

## 🚀 使用說明

### 線上體驗

本專案可部署於支援靜態網頁的任何平台：

- **Poe Canvas**：直接在 Poe 平台上運行
- **GitHub Pages**：免費託管
- **Netlify / Vercel**：一鍵部署

### 本地運行

```bash
# 克隆專案
git clone https://github.com/YOUR_USERNAME/luxun-second-brain.git

# 進入專案目錄
cd luxun-second-brain

# 使用任意 HTTP 伺服器運行（例如 Python）
python -m http.server 8000

# 或使用 Node.js
npx serve
```

然後在瀏覽器開啟 `http://localhost:8000`

> ⚠️ **注意**：對話功能需要在 Poe 平台環境中運行才能正常使用。本地運行僅能預覽介面。

## 📚 魯迅生平簡介

**魯迅**（1881年9月25日－1936年10月19日），原名**周樹人**，字**豫才**，浙江紹興人。

### 重要經歷

| 年份 | 事件 |
|------|------|
| 1902 | 赴日本留學，先學醫 |
| 1906 | 棄醫從文，立志以文學改造國民精神 |
| 1918 | 發表《狂人日記》，開創中國現代文學 |
| 1923 | 出版小說集《吶喊》 |
| 1927 | 定居上海，專事寫作 |
| 1936 | 病逝於上海 |

### 文學地位

- 中國現代文學的奠基人
- 白話文運動的主要推動者
- 對中國國民性有深刻批判與反思
