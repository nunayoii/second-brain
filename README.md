# 魯迅 · 第二大腦 (Lu Xun Second Brain)

## 簡介

結合作品資料庫與 AI 對話能力，將零散的魯迅文本與思想轉化為可檢索、可對話、可思考的「第二大腦」，讓閱讀不再停留於摘錄，而能形成系統性的理解與批判。

### 核心理念

> 「橫眉冷對千夫指，俯首甘為孺子牛。」——魯迅

---

## 功能特色

### 1. 作品資料庫

系統性整理魯迅部分主要的文學創作，包含：

| 類別 | 作品數量 | 代表作品 |
|------|----------|----------|
| 小說集 | 3 部 | 《吶喊》、《彷徨》、《故事新編》 |
| 散文集 | 2 部 | 《朝花夕拾》、《野草》 |
| 雜文集 | 12 部 | 《熱風》、《華蓋集》、《且介亭雜文》等 |
| 學術著作 | 2 部 | 《中國小說史略》、《漢文學史綱要》 |
| 翻譯作品 | 多部 | 果戈里《死魂靈》、法捷耶夫《毀滅》等 |

### 2. AI 驅動的對話系統

- **角色扮演**：AI 模型經過特殊提示詞工程，能夠模擬魯迅的語言風格、思維方式和價值觀  
- **上下文記憶**：支援多輪對話，系統會記住對話歷史，實現連貫的交流  
- **串流回應**：採用串流技術，即時顯示 AI 的回應過程
