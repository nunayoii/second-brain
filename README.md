# 魯迅 · 第二大腦 (Lu Xun Second Brain)

<div>

## 簡介

結合作品資料庫與 AI 對話能力，將零散的魯迅文本與思想轉化為可檢索、可對話、可思考的「第二大腦」，讓閱讀不再停留於摘錄，而能形成系統性的理解與批判。

### 核心理念

> 「橫眉冷對千夫指，俯首甘為孺子牛。」——魯迅

---

## 功能特色

### 1. 作品資料庫

系統性整理魯迅部分主要的文學創作，包含：

| 類別 | 作品數量 | 代表作品 |
|------|----------|----------|
| 小說集 | 3 部 | 《吶喊》、《彷徨》、《故事新編》 |
| 散文集 | 2 部 | 《朝花夕拾》、《野草》 |
| 雜文集 | 12 部 | 《熱風》、《華蓋集》、《且介亭雜文》等 |
| 學術著作 | 2 部 | 《中國小說史略》、《漢文學史綱要》 |
| 翻譯作品 | 多部 | 果戈里《死魂靈》、法捷耶夫《毀滅》等 |

### 2. AI 驅動的對話系統

- **角色扮演**：AI 模型經過特殊提示詞工程，能夠模擬魯迅的語言風格、思維方式和價值觀
- **上下文記憶**：支援多輪對話，系統會記住對話歷史，實現連貫的交流
- **串流回應**：採用串流技術，即時顯示 AI 的回應過程

### AI 整合

本專案整合使用 **Claude Sonnet 4.5** 作為對話引擎：

```javascript
    `@Claude-Sonnet-4.5 ${systemPrompt}\n\n${userMessage}`,
    {
        handler: 'luxun-handler',
        stream: true,        // 串流回應
        openChat: false      // 在應用內顯示
    }
);

## 魯迅生平簡介

**魯迅**（1881年9月25日－1936年10月19日），原名**周樹人**，字**豫才**，浙江紹興人。

### 重要經歷

| 年份 | 事件 |
|------|------|
| 1902 | 赴日本留學，先學醫 |
| 1906 | 棄醫從文，立志以文學改造國民精神 |
| 1918 | 發表《狂人日記》，開創中國現代文學 |
| 1923 | 出版小說集《吶喊》 |
| 1927 | 定居上海，專事寫作 |
| 1936 | 病逝於上海 |

### 文學地位

- 中國現代文學的奠基人
- 白話文運動的主要推動者
- 對中國國民性有深刻批判與反思
